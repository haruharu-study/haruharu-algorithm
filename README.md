# 하루하루 알고리즘 풀기

## 규칙

- *공휴일, 주말* 제외 **평일(월-금)** 알고리즘 문제를 풀고 올린다.
- 자정(24:00)까지 해당 주차에 파일을 push 한다.
  - 개인사정이 있을 경우, 단톡방에 이를 공지한다.
  - 잠수시 퇴출한다.
- 자바파일 이름의 양식은 `DayXXXX.java`으로 한다.
  - 하루에 여러 문제를 풀 경우에는 `DayXXXX-N.java` 형식으로 한다.
  - e.g. `Day0205.java`, `Day0205-2.java`
